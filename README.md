## Product Backlog

### Product Backlog – Project Part 2
| ID    | User Story                                                                                                                                                                                                      | Priority | Status |
|-------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------|--------|
| US-1  | As a new user, I want to create a new account so I can access the Counselling Appointment System.                                                                                                               | High     | Open   |
| US-2  | As a registered user, I want to log in so that I can access my account in the Counselling Appointment System.                                                                                                   | High     | Open   |
| US-3  | As a student, I want to browse counselor profiles and check their availability so I can find a suitable counselor.                                                                                              | High     | Open   |
| US-4  | As a student, I want to filter counselor profiles according to experience, specialization, gender, and whether they take in-person or online sessions so that I can find a counselor that meets my preferences. | Medium   | Open   |
| US-5  | As a student, I want to book a preferred time slot with a counselor so I can book my appointment.                                                                                                               | High     | Open   |
| US-6  | As a student or counselor, I want to receive notifications about upcoming appointments so I can remember to be present at them.                                                                                 | Medium   | Open   |
| US-7  | As a student or counselor, I want to be able to cancel existing appointments or reschedule them so I can avoid a no-show.                                                                                       | High     | Open   |
| US-8  | As a counselor, I want to be able to have an updateable schedule which is visible to students so that they can book appointments during my available times.                                                     | High     | Open   |
| US-9  | As a counseling office administrator, I want to be able to track and analyze no-shows so I can identify attendance issues.                                                                                      | Medium   | Open   |
| US-10 | As a student, I want to have access to a pre-session intake form that the counselor receives so that they have all relevant information before the session.                                                     | Medium   | Open   |
| US-11 | As a student, I want to be provided a post-session feedback mechanism that the counseling office administrators receive so that counselors can adjust and improve their practice.                               | Medium   | Open   |
| US-12 | As a student or counselor, I want to have a history of my past appointments visible to me so that I can review them.                                                                                            | Medium   | Open   |
| US-13 | As a counseling office administrator, I want to have a record of all student counseling appointments so that I can manage and oversee counselling services.                                                     | Medium   | Open   |
